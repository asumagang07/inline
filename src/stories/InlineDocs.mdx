An open source ui component library created with react, typescript and tailwind css that helps developer create flexible and reuseable ui components for their projects.

## Get Started

To use inline components just install inline components first.

```
yarn add inline-components
```

Then import the component you want to use.

```
import { Button } from "inline-components";

export const index = () => {
  return (
    <Button id='button' label='Button' />
  )
}
```
